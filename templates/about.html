À propos du Générateur d'Applications IA

Présentation du projet

Ce projet utilise l'IA pour générer des applications complètes et fonctionnelles basées sur les descriptions des utilisateurs. Il s'appuie sur l'API OpenRouter pour concevoir, générer et valider le code d'application. L'interface Flask offre un moyen intuitif de créer des applications sans écrire de code.

Prérequis

  • Python 3.8 ou supérieur

    L'environnement d'exécution principal pour le générateur d'applications

  • Pipenv

    Outil de gestion d'environnement virtuel et de dépendances

  • Clé API OpenRouter

    Nécessaire pour accéder aux modèles d'IA (Obtenir une clé)

Exemples d'utilisation

Pour de meilleurs résultats, essayez d'être spécifique dans vos descriptions. Voici quelques exemples :

Créer un site web portfolio statique pour un photographe avec une page d'accueil, galerie, à propos et contact. Inclure un design responsive au look moderne.

Construire une application de gestion de tâches avec authentification utilisateur. Les utilisateurs doivent pouvoir créer, modifier et supprimer des tâches, définir des dates d'échéance et marquer les tâches comme terminées.

Créer une API RESTful pour une plateforme de blog avec des endpoints pour les articles, commentaires et utilisateurs. Inclure l'authentification et la gestion appropriée des erreurs.
